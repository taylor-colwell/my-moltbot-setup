# Elisity Slide Layout Reference Guide

This guide documents all 45 slide layouts available in the Elisity master template. Use this to select the appropriate starting canvas for your slides.

## Quick Reference: Recommended Layouts for Manual Design

For building custom diagrams and arranging elements manually, prefer these **blank canvas** layouts:

| Layout | Name | Best For |
|--------|------|----------|
| **41** | Light_Headline_01 | Standard slides with eyebrow + headline |
| **42** | Light_Headline_02 | Alternate headline style |
| **36** | Light_No-Headline | Full canvas, no header |
| **37** | Light_No-Headline_2 | Full canvas variant |
| **38** | 1_Light_No-Headline_2 - NO BAR | Full canvas, no sidebar |
| **39** | Light_Blank | Completely blank |
| **43** | Light_Blank_01 | Blank with basic structure |

## Layout Selection Guide

### For Custom Diagrams (Manual Element Arrangement)
**Use:** Layout 41 (Light_Headline_01) or Layout 36-39 (blank variants)
- Provides clean canvas
- Headline placeholder for slide title
- Maximum space for custom element arrangement

### For Text-Heavy Content
**Use:** Layouts 1-5, 15-17
- Pre-defined text placeholders
- Automatic formatting
- Good for bullet points and paragraphs

### For Data/Statistics
**Use:** Layouts 12-14
- Big number displays
- Stat callouts
- Data visualization backgrounds

### For Architecture Diagrams
**Use:** Layout 41 + manual elements OR Layout 8-9 (Title+Blank)
- Title/header for context
- Large blank area for diagram construction
- Import icons from element library

### For Comparison/Multi-Topic
**Use:** Layouts 3-7, 29-30
- Pre-defined column structures
- Even spacing
- Good for side-by-side comparisons

## Workflow: Using Layouts with Manual Element Arrangement

1. **Select layout** (typically 41 or blank variant)
2. **Set headline/eyebrow** if using headline layout
3. **Import elements** from iconography library
4. **Arrange manually** on the canvas
5. **Add connectors/lines** as needed
6. **Add text labels** for elements

This approach gives maximum flexibility while maintaining brand consistency through the master template's colors, fonts, and footer elements.
