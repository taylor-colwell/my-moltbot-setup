## Usage Workflow

### Using SVG Icons (Recommended)
1. Rasterize SVG to PNG using Sharp:
   ```javascript
   const sharp = require('sharp');
   await sharp('icons/virtual-edge-node.svg')
     .resize(200, 200)
     .png()
     .toFile('workspace/virtual-edge-node.png');
   ```
2. Insert PNG into slide using OOXML or html2pptx

### Using Element Library (Copy/Paste)
1. Open `templates/elisity-element-library.pptx`
2. Navigate to the appropriate slide (1-9)
3. Select and copy desired element
4. Paste into target slide
5. Resize and position as needed

### Building Diagrams
1. Start with Layout 41 (Light_Headline_01) or blank layout
2. Set eyebrow and headline text
3. Import/paste required elements
4. Arrange elements spatially
5. Add connectors/lines between elements
6. Add text labels as needed
7. Apply consistent sizing (typically 50-80px for icons)

## Common Diagram Patterns

### Network Topology
- Cloud providers (top) -> Virtual Edge (middle) -> Endpoints (bottom)
- Use arrows/lines to show data flow
- Group devices by location or function

### Architecture Diagram
- CCC at center or top
- VE nodes distributed across sites
- VENs at switch layer
- Devices at edge

### Policy Flow
- Identity sources (left) -> Policy engine (center) -> Enforcement (right)
- Use policy icons to show rules
- Security profiles at decision points

### Site Layout
- Location icon as container
- Infrastructure icons within
- Device groups at bottom
- Cloud connection at top
